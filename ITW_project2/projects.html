Projects that I have created

Black Books

This was the first project in the ITW (Web Development) university course. The assignment focused on recreating an existing website layout using only the provided HTML structure — modification of HTML files was strictly forbidden. This constraint allowed for deep practice with CSS selectors, specificity, and layout techniques. Key learning outcomes:

  • Advanced CSS selectors and combinators to style existing markup
  • CSS advanced selectors
  • Application of box model, positioning, shadows, gradients, grids and etc

The final product accurately mimics the aesthetic and functionality of the original design, including text alignment, fonts, icons and playful layout elements — all done without touching the underlying HTML.

Weather Forecast Web

This is a responsive weather forecast web application built using HTML, CSS, and JavaScript. The user can enter any city to view a 6-day weather forecast, including temperature highs and lows, and weather conditions represented by icons (e.g., sun, clouds, rain, snow, wind).

  • City search: City search: Type a city and click "Change city" to fetch and display the forecast.
  • 6-day outlook: Displays daily weather with minimalistic icons and temperature details.
  • Live time:Displays current day and time in real-time.

This project demonstrates DOM manipulation and responsive layout design using vanilla JavaScript and CSS.
